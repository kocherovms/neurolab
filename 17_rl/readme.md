**КОНТЕКСТ**

Хочется развить и проработать идею о мысле-векторном устройстве мозга (Thought-Vector). Есть многообещающий опыт работы с трансформерами, трансформеры удачно воплощают идеи о том, как наш мозг обрабатывает информацию. Задачи RL (Reinforcement Learning) совпадают с теми, которые решает наш мозг. 

**ИДЕЯ**

Прикрутить трансформеры к задачам RL (Reinforcement Learning). Сделать трансформер - мозгом агента.

**ЗАДАЧИ**

1) поиграться с различными алгоритмами RL из арсенала CleanRL: PPO, DQN
2) разработать болванки ноутов, интегрировать с инфраструктурой (nexus, tensorboard, ...)
3) отработать вопросы масштабирования (векторные окружения)
4) разработать агента на базе транформера
